## Running in a Dev Environment:
```gradle
gradlew setupDecompWorkspace
gradlew runClient
```

## Compiling the client:
```gradle
gradlew setupDecompWorkspace
gradlew build
```
You will find your built jar in the directory `builds/libs`.
